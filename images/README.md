# Capturas del dashboard

Esta carpeta esta destinada a guardar las imagenes exportadas desde Power BI Desktop una vez que el dashboard sea construido manualmente.

## Capturas recomendadas

- `dashboard_general.png`: vista completa del dashboard.
- `kpis_principales.png`: tarjetas con Ventas Totales, Cantidad Vendida, Numero de Ventas y Ticket Promedio.
- `ventas_por_mes.png`: grafico de lineas con la evolucion mensual de ventas.
- `ventas_por_categoria.png`: grafico de barras por categoria.
- `top_productos.png`: grafico con los 5 productos mas vendidos.
- `ventas_por_zona.png`: grafico comparativo de ventas por zona.

## Recomendaciones

- Usar nombres de archivo claros.
- Mantener las capturas en formato PNG.
- Verificar que los textos y valores sean legibles.
- Actualizar el README principal cuando se agreguen capturas reales.
